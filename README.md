# 📚 Edura — AI-Powered Study Platform

**Edura** is an EdTech prototype designed to make education more affordable, accessible, and engaging for students. This platform empowers students with AI-driven tools, gamified study rewards, and institutional integration for personalized learning support.

---

## 🚀 Features

- 📖 **Textbook Access** — Affordable textbook downloads via subscription or point-based system
- 🔁 **Book Trading** — Trade textbooks with other students to save money and reduce waste
- 🤖 **AI Study Tools** — Generate flashcards, get concept breakdowns, and create group quizzes
- 🎓 **Institution Hub** — Dashboard for educators to monitor progress and manage resources
- 🎮 **Gamification** — Earn points, rewards, and achievements for consistent studying
- 🔐 **Multiple Logins** — Use Google, Email/Phone, or Institution ISO to sign in

---

## 🧪 Subscription Tiers

| Plan         | Features Highlights |
|--------------|---------------------|
| **Freemium** | Ad-supported, previews, basic access |
| **Student**  | 4 downloads/month, avatar, basic homework help |
| **Pro**      | 10 downloads, AI tools, textbook sharing, group calls |
| **Platinum** | All Pro + .edu tools, referrals, daily login rewards |

## 🛠️ Tech Stack

- **React + Tailwind CSS** (Frontend UI)
- **React Router** (Navigation)
- **Vite** (Build tool)
- **Component-based structure** with reusable UI modules

---

## 📂 Folder Structure

```
src/
├── components/
│   ├── ui/            # Reusable UI elements (buttons, cards, inputs)
│   └── BottomNav.jsx  # Navigation footer
├── pages/             # Each screen of the Edura app
├── App.jsx            # Routes and layout
├── index.css          # Tailwind entry
└── main.jsx           # App entry point
```

## 🧠 Getting Started (Dev)

1. Clone or fork this repo in StackBlitz or your local IDE
2. Make sure dependencies are installed in `package.json`
3. Start your dev server:
   ```bash
   npm run dev
   ```
